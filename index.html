Technologies used

Written in HTML, Javascript and CSS (or actually, LESS). I used CodeKit for compiling the LESS source and minifying the CSS/JS files.

Writing everything yourself is silly when you can use these great open source libraries:

  • Twitter Bootstrap as a base for styling and stuff,
  • jQuery (I don't know how people did it before?!),
  • FancyBox to display my instagrams in a lightbox,
  • Moment.js for time formatting.

Because having a site that updates itself is totally awesome, I used these lovely APIs:

  • Last.FM API
  • ThisIsMyJam API
  • Instagram API
  • Github API
  • The Twitter API requires authentication for all requests since mid June 2013, which can't be done (securely) in Javascript alone. I used this script instead.
